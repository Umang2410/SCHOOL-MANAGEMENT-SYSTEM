## About The Project

Student Management System is an essential system that provides a platform for managing the admission process for an educational  institution. This project aims to develop a web-based admission  management system using PHP, HTML, and MySQL database server. The  system is designed to manage two pages, an admin page and a student  page. The admin page is responsible for adding, deleting, modifying, and  searching student records, while the student page is responsible for  modifying and viewing their personalÂ information.
